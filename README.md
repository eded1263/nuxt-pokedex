# nuxt-pokedex
## A fanmade pokedex to learn more about NuxtJS

Tecnologies used:
- Vue
- Nuxt
- Typescript
- Vuex module system
- Vue class components and Vue property decorator
- Infinite Scrolling
- Scss
## Build Setup

```bash
# Bringing up the project
$ yarn build
$ yarn start

```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
